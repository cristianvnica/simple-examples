Cleaning Git Untracked Files Example

Here are some simple Git commands examples to help you get started with Git:

1. Initialize a new Git repository

git init

2. Clone an existing Git repository

git clone https://github.com/username/repository.git

3. Check the status of your local repository

git status

4. Add files to the staging area

Add single file:

git add file_name.ext

Add all files in the directory:

git add .

5. Commit changes to the repository

git commit -m "Your commit message here"

6. Connect your local repository to a remote repository

git remote add origin https://github.com/username/repository.git

7. Push your changes to the remote repository

Push a specific branch:

git push origin branch_name

To push the master branch:

git push origin master

8. Pull changes from the remote repository

git pull origin branch_name

9. Create a new branch and switch to it

git checkout -b new_branch_name

10. Switch to an existing branch

git checkout branch_name

11. Merge changes from one branch into another

git merge source_branch_name

12. Delete a local branch

git branch -d branch_name

13. View commit history

git log

These are just a few basic Git commands to get you started. There are many more commands and options you can use to manage your Git repositories.
