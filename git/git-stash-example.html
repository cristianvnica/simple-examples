Git Stash Example

Let's say you did some work, and you want to save it, so you can work on something else. Than in your terminal you can:

git stash save "my work in progress message"

Than work on something else, and reaply your changes with

git stash apply

when you're ready to continue.

You can browse your recent stash list with

git stash list

and apply an older stash by mentioning its number in the apply command:

git stash apply [number]
