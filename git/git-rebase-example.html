Git Rebase Example

Here is a simple example on how to perform 'git rebase'

git rebase --pull origin master

'git rebase --pull origin master' fetches the latest changes from the origin repository's master branch and then rebases the current branch on top of those changes. This helps to incorporate the latest changes from origin/master into the current branch with a linear commit history.
