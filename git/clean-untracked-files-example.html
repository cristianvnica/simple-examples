Cleaning Git Untracked Files Example

Let's assume you have a Git repository with the following structure:

my-repo/
  .git/
  file1.txt
  file2.txt
untracked-file.txt

In this example, file1.txt and file2.txt are tracked files, while untracked-file.txt is an untracked file. To delete the untracked file, follow these steps:

  1. Open the terminal and navigate to the my-repo directory: cd path/to/my-repo
  2. Run git status to see the current status of your repository
  3. Preview which untracked files will be removed with the git clean -n command
  4. If you're satisfied with the list of untracked files to be removed, proceed with the actual deletion by running: git clean -f

Now, the untracked file untracked-file.txt has been removed from your repository. Remember to use git clean with caution, as it permanently deletes untracked files.
