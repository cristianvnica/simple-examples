Index.html example

Here is a simple Index.html example.

<!DOCTYPE html>
<html lang="en">
  <head>
    <title>Page title</title>
  </head>
    
  <body>
    <header>
      <h1>Title</h1>
    </header>
    <nav>
      <p>Navigation</p>
      <a href="#">Page 1</a>
      <a href="#">Page 2</a>
    </nav>
    <main>
      <p>Content</p>
    </main>
    <footer>
      <p>Footer</p>
    </footer>
  </body>
</html>

The above HTML document can be styled using CSS (Cascading Style Sheets), and result in a web page looking like this:
