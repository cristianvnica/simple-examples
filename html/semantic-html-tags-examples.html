Semantic HTML Tags Examples

Semantic HTML refers to using specific HTML elements that convey meaning about the structure of a webpage, making it more accessible and easier to understand for both humans and machines. Here are some examples of semantic HTML elements and how to use them in a simple webpage:

  1. <!DOCTYPE html>: Declares the document type and version of HTML being used.
  2. <html>: The root element of the page.
  3. <head>: Contains meta-information, links to stylesheets, and other data not directly displayed on the page.
  4. <title>: Sets the title of the webpage, displayed in the browser's title bar or tab.
  5. <body>: Contains the content of the webpage.
  6. <header>: Contains the header section of a webpage, including logos, site navigation, and other introductory elements.
  7. <nav>: Contains navigation links, typically within a header or sidebar.
  8. <main>: Contains the main content of the webpage.
  9. <article>: Represents a self-contained piece of content, such as a blog post or news article.
  10. <section>: Represents a thematic grouping of content, typically with a heading.
  11. <aside>: Contains content that is tangentially related to the content around it, such as a sidebar or pull quote.
  12. <footer>: Contains the footer section of a webpage, including copyright information, contact details, and other site-wide information.

Here's an example of a simple webpage using semantic HTML:

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Semantic HTML Example</title>
</head>
<body>
  <header>
      <h1>My Website</h1>
      <nav>
        <ul>
          <li><a href="/">Home</a></li>
          <li><a href="/about">About</a></li>
          <li><a href="/contact">Contact</a></li>
      </ul>
        </nav>
    </header>
    <main>
        <article>
            <h2>Blog Post Title</h2>
            <p>Published on May 9, 2023</p>
            <p>Here is the content of the blog post...</p>
        </article>
        <section>
            <h2>Related Posts</h2>
            <ul>
                <li><a href="/post-1">Post 1</a></li>
                <li><a href="/post-2">Post 2</a></li>
            </ul>
        </section>
    </main>
    <aside>
        <h2>About the Author</h2>
        <p>Short bio of the author...</p>
    </aside>
    <footer>
        <p>Copyright &copy; 2023 My Blog Example. All rights reserved.</p>
    </footer>
</body>
</html>

This example demonstrates the use of various semantic HTML tags to create a simple webpage with a header, navigation, main content, related content, an aside, and a footer.
